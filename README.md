# SvartJack
## Prosjektoppgave TDT4100 - Objektorientert programmering

SvartJack er en Blackjack app der brukere kan spille blackjack mot en dealer. Brukerene skriver inn navn for å bli med på bordet og kan forlate når de er ferdige eller blakke. Når spillere forlater bordet blir navnet og brukerens saldo lagret i fil så de kan hente fram saldoen når de vil spille på nytt. Appen har klassisk Blackjack logikk, men mangler funksjonene split og double.
